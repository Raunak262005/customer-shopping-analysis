# 🛍️ Customer Shopping Behavior Analysis

A complete Data Analytics project that explores customer shopping behavior using **Python, Pandas, MySQL, and Power BI**. The project focuses on data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard creation to uncover valuable customer insights.

## 🚀 Project Highlights

* Cleaned and preprocessed raw customer shopping data
* Performed Exploratory Data Analysis (EDA)
* Wrote SQL queries to answer business questions
* Built an interactive Power BI dashboard
* Generated business insights from customer purchase behavior

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* MySQL
* Power BI
* Jupyter Notebook

## 📊 Analysis Performed

* Data Cleaning
* Missing Value Handling
* Gender-wise Revenue Analysis
* Age Group Analysis
* Purchase Frequency Analysis
* Category-wise Sales Analysis
* Customer Behavior Analysis

## 📁 Project Files

* `customer_shopping_behavior.csv` – Dataset
* `customer_shopping_behavior_analysis.ipynb` – Python Analysis
* `customer_shopping_behavior_queries.sql` – SQL Queries
* `Customer_Shopping_Dashboard.pbix` – Power BI Dashboard
* `dashboard.png` – Dashboard Screenshot
